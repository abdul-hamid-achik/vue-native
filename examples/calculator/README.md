# Calculator

An iOS-style calculator built entirely with Vue Native.

## What It Demonstrates

- **Components:** VView, VText, VButton
- **Patterns:** Computed display formatting, button grid layout, `v-for` rendering, dynamic inline styles

## Key Features

- Full arithmetic operations (+, -, x, /)
- AC, +/-, and % function keys
- iOS-native calculator look and feel
- Overflow handling with scientific notation

## How to Run

```bash
bun install
bun run dev
```
